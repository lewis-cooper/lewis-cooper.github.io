Command reference

Everything you can type in Todo Terminal.

This page is the full reference for the macOS app. Todo commands work on the current list, while app commands manage windows, named lists, theme, and highlighting.

Install flow: download the DMG, drag the app into Applications, then first-launch it with Right-click > Open or System Settings > Privacy & Security > Open Anyway.

How to read this page

Commands are shown exactly as typed. Aliases are grouped on the same line to keep the reference short.

Todo commands

Current list actions.

add <task>

Add one item. You can also separate multiple tasks with | in a single prompt.

edit <number> <entry>

Replace the text of an existing item.

remove <number...> rm <number...>

Delete one or more items.

move <number> <destination> mv <number> <destination>

Move an item to a new position.

clear

Remove everything from the current list.

help quit

Open the in-app command reference or close the app.

App commands

Lists, theme, and focus.

new <name> open <name>

Create a new named list or open one that already exists.

del <name> delete <name> del .

Delete a saved list by name, or delete the current list with del ..

close

Close the current window.

theme <light|dark>

Switch between the built-in light and dark palettes.

highlight <number> hl <number>

Highlight an item for focus.

Keyboard shortcuts

Font size.

Cmd +

Increase the terminal font size.

Cmd -

Decrease the terminal font size.

Tip: the app also includes a built-in command reference dialog, so typing help is the fastest way to check syntax from inside the app.

Todo Terminal keeps the command surface intentionally small so it stays easy to remember.
